# International Football Match Predictor

A machine learning project that predicts outcomes of international football matches using historical match data.

## Features
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Random Forest Classification
- Streamlit Web Application

## Dataset
49,000+ international football matches.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Results
Accuracy: ~54%

## Run Locally

pip install -r requirements.txt

streamlit run app.py
